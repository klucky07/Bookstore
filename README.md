
# Library

a simple Library website to implement CRUD operations


## Run locally

To run this project fork the project and clone it to your local machine and then follow the commands.

for frontend

```bash
  cd frontend
  npm run dev 
```
for backend

```bash
 cd backend 
 node index.js


## Documentation
